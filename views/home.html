What is ComposeUp?

ComposeUp is a browser based software, create to manage important documentation like Pay Order Letter, Pay Order Cancellation and many other things, but the main feature is GST Return section which is dynamically calculate GDs information and create monthly Summary frequently by just few clicking.

This software is created by the combination of many technologies, list of these technologies are as following.
  • HTML5 for creating structure)
  • CSS3 (use for visibility and all the major graphic)
  • Bootstrap (use for template like font and table styling)
  • JavaScript (the main programming language for this software is JavaScript)
  • JOSN (use to creating accounts listing)
  • AngularJS (it's JS library, use to manage routing system, controllers, modules, services etc.)
  • PHP5 (this is use to make it server side for the security purpose, and also connecting to database)
  • MySQL (this is main Database to holds all data you have created)
